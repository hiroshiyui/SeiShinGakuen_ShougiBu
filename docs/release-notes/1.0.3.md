# 1.0.3

二件のフリーズ系バグ修正。

## Fixed

- AI が合法手なし (玉一枚で全逃げ場封鎖など) で固まる問題を解消。
  `手詰まり` を検出して相手勝ちで終局するように。
- Android のバックジェスチャがアプリを直接終了させていた問題。
  Esc キーと同じ入力経路を経由するように合成イベントを差し替え、
  対局画面・棋譜画面などから期待どおりタイトルへ戻るように。
