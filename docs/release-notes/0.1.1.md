# 0.1.1 — 最初の公開リリース

**清正学園将棋部** の最初の公開リリースです。

## ハイライト

- AlphaZero スタイルの AI 対局相手を**端末内**で実行（オンライン接続不要）
- 強さプリセット **Lv 1〜Lv 8** に 8 名の AI キャラクター（佐藤竜太郎・鈴木すず・高橋ゆり子・伊藤明・中村アリス・テリー・クラーク・吉田なな・加藤よしこ）
- **先生モード** — MCTS による上位 3 手の候補を勝率 % 付きで表示
- 本将棋の合法手判定をフル実装（王手・二歩・打ち歩詰め・千日手）
- 駒のスライドアニメーションと将棋盤のズーム演出

## 動作環境

- Android arm64-v8a。Mobile レンダラ（Vulkan）。
- ネットワーク権限なし — すべての推論は端末内の `bonanza.onnx`（1.3 MB）に対して `tract` で実行。

## 同梱ファイル

- `seishingakuen-release-v0.1.1.apk` — 署名済みリリース APK（サイドロード用）

Google Play への AAB 公開は次回以降のリリース予定。
