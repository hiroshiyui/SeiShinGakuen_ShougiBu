# 0.2.0 — 対戦相手選択画面とキャラクター紹介

## Highlights

- **対戦相手選択画面** — 8 人の部員と先生から指す相手を選べるようになりました。各キャラクターには肖像画と紹介文が用意されています。
- **対局画面の対戦相手バー** — 上部に対戦相手の肖像と名前を表示。誰と指しているのかが一目で分かります。
- **キャラクター紹介の充実** — 加藤師範、吉田顧問、中村部長、鈴木副部長、高橋・伊藤・テリー・佐藤の各部員に背景設定と肖像画が追加されました。
