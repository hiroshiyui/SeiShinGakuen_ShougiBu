# 0.1.2 — 背景画像の差し替え

**0.1.1 のパッチリリース**。背景画像の差し替えと、それに合わせたアイコン・スクリーンショットの更新が中心。

## 変更点

- **背景画像を本来意図していた教室シーンに修正**（0.1.1 では誤って別の画像が同梱されていた）。タイトル画面・対局画面の両方が、絵画と本棚のある教室で将棋盤を囲む構図に。
- **アプリアイコンを刷新** — 新しい背景の机上の将棋盤を 1:1 でクロップしたデザインに変更。Android アダプティブアイコン（192×192 メイン / 432×432 フォアグラウンド）も同時に更新。
- **README のスクリーンショットを差し替え** — 修正後の背景での新規キャプチャ（タイトル画面・初期局面・先生モードの提案）に更新。
- 内部: `docs/screenshots/` に `.gdignore` を配置。Godot のインポート対象から除外し、`.import` 補助ファイルがリポジトリに混入するのを防ぐ。

## 同梱ファイル

- `seishingakuen-release-v0.1.2.apk` — 署名済みリリース APK（サイドロード用）

Google Play への AAB 公開は引き続き次回以降のリリース予定。
