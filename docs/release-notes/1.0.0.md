# 1.0.0 — 安定版リリース + 隠しクレジット画面

## Highlights

- **1.0 として安定版をリリース。** Phase 1〜7 の全項目を完了。完全な本将棋ルール (王手・二歩・打ち歩詰め・千日手・入玉宣言・持将棋)、AlphaZero 系オフライン AI 対局 (Lv 1〜8 の 8 キャラクター)、棋譜の保存と検討 (MCTS による好手 / 疑問手 / 悪手の自動評価)、先生モードによる候補手提示が、Linux デスクトップでの開発実行と Android arm64-v8a 端末でのサイドロード配布に対応した状態で揃いました。
- **隠しクレジット画面。** タイトル「清正学園将棋部」を 3 秒以内に 10 回タップすると、作者より + AI モデル (Bonanza / ShogiDojo / tract) + エンジン (Godot, godot-rust) + フォント謝辞のクレジット画面が開きます。隠す ボタンで本文パネルを畳み、背景アートだけを眺めることもできる小ネタつき。

## Changes

### Features

- `feat(menu)` タイトル 10 回タップ (3 秒以内のローリングウィンドウ) で `scenes/Credits.tscn` を解錠。作者より、AI モデル謝辞、エンジン謝辞、フォント謝辞 (Noto Serif JP / M PLUS 2 / 筆ごしらえ — いずれも SIL OFL 1.1) を表示 (924b40a)
- `feat(credits)` クレジット画面に 隠す ボタンを追加。ヘッダーと本文を畳んで背景イラストのみ表示。画面のどこかをタップすると復帰、Android のバックジェスチャは「畳んでいたら戻す → 畳まれていなければタイトルへ」の二段挙動 (89d05a0)

### Fixes

- `fix(menu)` Android では 1 タップにつき `InputEventScreenTouch` と合成された `InputEventMouseButton` の両方が飛んでくるため、タイトルタップが 2 重カウントされて 5 回でクレジット画面が開いていた。`Square.gd` と同じ `OS.has_feature("mobile")` ベースの分岐に揃え、モバイルはタッチ、デスクトップはマウスのみを購読するよう修正 (6a6087b)
