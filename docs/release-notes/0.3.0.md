# 0.3.0 — 棋譜検討と入玉宣言

## Highlights

- **棋譜検討モードを追加。** タイトル画面の「棋譜検討」から、過去の対局を保存した KIF ファイルを開いて再生できます。`<<` `<` `>` `>>` で局面を進めるだけでなく、「解析」ボタンを押すと全手を MCTS が読んで、各手の先手勝率と分類バッジ（◎好手 / △疑問手 / ×悪手）、さらに疑問手以下なら推奨手まで表示します。対局中の「棋譜」パネルからは「保存」で `.kif` を端末の Documents フォルダに書き出せます。
- **入玉宣言ルールに対応。** 玉が敵陣に入ると「入玉宣言」ボタンが現れ、公式の 28点（先手）/ 27点（後手）法と敵陣 10 枚以上の駒数条件を満たせば宣言勝ち。条件不足のときは何が足りないかを画面で案内します。AI 側も自動で宣言、両者同時に条件を満たせば持将棋（引き分け）。
- **UI 一新。** ダーク × ゴールドのボタンとダイアログ、加藤先生の丸アイコン、設定画面（効果音オン/オフ・先生ボタン位置）。Android のバックジェスチャは投了確認・画面戻りに正しく繋がるようになりました。
- **ライセンスを GPL-3.0-or-later に決定。** 0.1.x / 0.2.x まで「ライセンス未指定」だった本プロジェクトを、本リリースから **GNU General Public License v3.0 以降** で配布します。詳細は [`LICENSE`](https://github.com/hiroshiyui/SeiShinGakuen_ShougiBu/blob/0.3.0/LICENSE) と [ADR-0010](https://github.com/hiroshiyui/SeiShinGakuen_ShougiBu/blob/0.3.0/docs/adr/0010-gpl-3-or-later.md) を参照。
