# 1.0.1 — バージョン表示と安全領域の統一

## Highlights

- **メインメニュー右下にバージョン表示。** タイトル画面の右下に M PLUS 2 Regular で `v1.0.1` を表示。`application/config/version` を `project.godot` に追加し、画面表示の単一の出典に。
- **`SafeAreaLayout` ドロップインスクリプトに集約。** 設定画面・棋譜検討・棋譜一覧・クレジット画面・タイトル画面の上下インセットを、ステータスバーやジェスチャーナビ領域からきちんと避けるよう共通スクリプトに統一。新しいフルスクリーン画面はフルレクト Control に `scripts/SafeAreaLayout.gd` をアタッチするだけで `_apply_safe_area` のボイラープレートがなくなる。
