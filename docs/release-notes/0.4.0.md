# 0.4.0 — 定跡データベースと棋譜閲覧の改善

## Highlights

- **定跡データベース (Opening Book) を追加。** 序盤 4〜5 手の主要な戦型 (相居飛車・矢倉系・中飛車・四間飛車) を ~12 手分の定跡データとして同梱。AI は対局開始時にまず定跡を参照し、ヒットすればそのまま指し、外れれば従来通り MCTS で読みに切り替えます。同じキャラクターでも序盤に変化が出やすくなり、思考時間も短縮されます。
- **棋譜検討の盤面に最終手ハイライト。** 対局中と同じ青い半透明オーバーレイで「いま動いた駒」が分かるようになりました。`<<` `<` `>` `>>` で局面を進めると追従します。
