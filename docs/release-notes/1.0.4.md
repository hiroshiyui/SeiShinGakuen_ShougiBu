# 1.0.4

Android 体験の安定化。

## Fixed

- リリース APK でバックジェスチャがアプリを直接終了させていた問題。
  1.0.3 の入力合成方式がリリースビルドで取りこぼされていたので、
  各画面で `WM_GO_BACK_REQUEST` を直接処理する方式に置き換え。

## Changed

- 対局終了ダイアログに「閉じる」ボタンを追加。再戦せずに終局画面
  を眺めたい時の離脱経路を確保。
- ダイアログ右上の小さい × アイコンをテーマで非表示。タップ判定が
  小さく、見た目の位置と当たり判定がずれる Godot 4 の制約も合わせて
  回避。OK / キャンセル / Android バックで閉じる。
